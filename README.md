# Royal Mahira Group Website

This project is a static website for Royal Mahira Group with multiple pages for Home, About, Products, Services, and Contact.

## Overview
- Responsive layout for desktop, tablet, and mobile.
- Mobile-friendly sidebar navigation menu on all main pages.
- Clean sections for company information, products, services, and contact details.

## Pages
- index.html — Home page
- about.html — About page
- products.html — Products page
- services.html — Services page
- contacts.html — Contact page

## Mobile Navigation Update
All main pages now include:
- a compact hamburger menu on small screens,
- a slide-in sidebar navigation panel,
- overlay behavior for closing the menu,
- responsive spacing to prevent the menu from covering the page body.

## How to View the Site
1. Open the project folder in a browser.
2. Launch index.html directly, or use a simple local server if preferred.

## Notes
- The website uses plain HTML, CSS, and a small JavaScript snippet for the mobile menu.
- No build tool is required for normal viewing.

## Maintenance
If you update the pages, keep the same responsive navigation pattern on all pages for a consistent mobile experience.
